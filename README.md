# 🤖💰 AI Budget & Deals Assistant for Latvia

**Your Smart Financial Companion - Jūsu viedais budžeta asistents - Ваш умный помощник по бюджету**

A revolutionary AI-powered budget tracking and deals finder app specifically designed for the Latvian market. Track your spending, find the best deals from local stores, get discount codes, and chat with an intelligent AI assistant that helps you save money!

![Multi-Language](https://img.shields.io/badge/Languages-LV%20%7C%20RU%20%7C%20EN-blue)
![Latvia Focus](https://img.shields.io/badge/Market-Latvia%20🇱🇻-red)

## ✨ **Features**

### 🤖 **AI Chat Assistant**
- **Smart Conversations**: Natural language processing in Latvian, Russian, and English
- **Budget Queries**: "How much can I spend today?" / "Cik es varu tērēt šodien?" 
- **Meal Planning**: "Find me a 5€ meal" with real store prices
- **Deal Alerts**: Get notified about the best discounts
- **Multi-language Support**: Seamless switching between languages

### 💰 **Budget Tracking**
- **Daily/Weekly/Monthly Budgets**: Set and track your spending limits
- **Expense Categories**: Food, Transport, Shopping, Entertainment, Bills, Other
- **AI Insights**: Smart recommendations to save money
- **Visual Analytics**: Beautiful charts and progress bars
- **Overspending Alerts**: Get warned before you exceed your budget

### 🛒 **Deals Finder**
- **Real-time Price Scraping**: From Maxima, Rimi, Barbora, Citro (coming soon)
- **Best Deal Alerts**: Automatically find the lowest prices
- **Store Comparison**: Compare prices across multiple retailers
- **Category Filtering**: Focus on specific product types
- **Price History**: Track how prices change over time

### 🎫 **Discount Coupons**
- **Active Coupon Tracking**: Current discount codes from Latvian stores
- **Automatic Verification**: Check if codes are still valid
- **Copy & Use**: One-click coupon copying
- **Expiry Monitoring**: Never miss a deal again
- **Store-specific Codes**: Organized by retailer

### 🎨 **Beautiful Modern UI**
- **Glassmorphism Design**: Modern frosted glass effects
- **Dark/Light Theme**: Choose your preferred appearance
- **Mobile-First**: Optimized for smartphones and tablets
- **Smooth Animations**: Delightful micro-interactions
- **Accessible**: Designed for all users

## 🚀 **Technology Stack**

### **Frontend**
- **React 18** with TypeScript
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **Lucide React** for icons
- **React Router** for navigation

### **Internationalization**
- **react-i18next** for multi-language support
- **Browser language detection**
- **Persistent language selection**

### **State Management**
- **React Hooks** (useState, useEffect, useCallback)
- **LocalStorage** for data persistence
- **Context API** for global state

### **Future Integrations**

## 🏁 **Quick Start**

### **Prerequisites**
- Node.js 14+ 
- npm or yarn

### **Installation**

```bash
# Clone the repository
git clone https://github.com/knoticoo/Strategy.git
cd Strategy

# Install dependencies
npm install

# Start development server
npm start
```

The app will open at `http://localhost:3000`

### **Build for Production**

```bash
# Create production build
npm run build

# The build folder contains the optimized app
```
